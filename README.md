# Retail-Analytics-Customer-Behavior-and-RFM-Analysis-
This project presents a complete Retail Analytics solution built using Excel (CSV), SQL, Python, and Power BI to analyze customer purchasing behavior and implement RFM (Recency, Frequency, Monetary) segmentation.
The goal of this project is to convert raw retail transaction data into meaningful business insights that support customer retention strategies, revenue optimization, and data-driven marketing decisions.
